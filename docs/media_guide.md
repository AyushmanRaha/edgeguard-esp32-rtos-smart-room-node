# Media Capture Guide

Use this guide when adding future repository images or a release demo video.

| Asset | Suggested capture |
| --- | --- |
| `media/edgeguard_hero_build.jpg` | Wide photo of the complete low-voltage prototype on a desk or breadboard. |
| `media/edgeguard_wiring_closeup.jpg` | Close view with ESP32 GPIO wiring, HC-SR04 divider, DHT11, LDR, relay inputs, and LEDs visible. |
| `media/edgeguard_dashboard_status.png` | Browser screenshot of the dashboard with mode, state, sensors, relays, IP, heap, and event log. |
| `media/edgeguard_serial_monitor.png` | Serial monitor screenshot showing boot, Wi-Fi or AP address, and periodic sensor logs. |
| Demo video | Short release asset showing boot, dashboard, AUTO behavior, AWAY alert, and fail-safe relay state. |

## Safety and privacy checks

- Show low-voltage DC wiring only.
- Do not show Wi-Fi passwords, private network names, tokens, or personal information.
- Keep labels readable and wiring uncluttered.
- Verify the dashboard is on a trusted local network or fallback AP before recording.
